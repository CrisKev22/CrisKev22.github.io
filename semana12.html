Desarrollo backend - Desarrollar Aplicaciones con Spring Core y Spring Boot

Definiciones


1. Arquitectura de aplicaciones web: Estructura y diseño de aplicaciones web que incluye el frontend, backend, bases de datos, y la comunicación entre estos componentes.


2. Servidores web: Software que sirve contenido web (páginas HTML, archivos, datos) a los clientes a través de Internet o una red local.


3. Funcionamiento del server side: Procesos y operaciones que se realizan en el servidor para gestionar solicitudes y respuestas, manipular bases de datos, y ejecutar lógica de negocio.


4. Lenguaje y framework backend: Lenguajes de programación y frameworks utilizados para desarrollar la lógica del servidor. Ejemplos incluyen Java con Spring y Jakarta.


5. Java y Jakarta: Java es un lenguaje de programación ampliamente utilizado para el desarrollo backend. Jakarta es una colección de especificaciones para construir aplicaciones empresariales en Java.


6. IntelliJ IDEA: Entorno de desarrollo integrado (IDE) para la programación en varios lenguajes, especialmente Java.


7. Tomcat: Servidor web de código abierto que ejecuta servlets y JSP (JavaServer Pages) para aplicaciones web basadas en Java.


8. Spring Core y Spring Boot: Frameworks de Java para el desarrollo de aplicaciones empresariales. Spring Core proporciona la infraestructura básica, mientras que Spring Boot facilita la creación de aplicaciones independientes y listas para producción.


Procedimientos


1. Desarrollo backend con Java y Spring:

  • Configurar un proyecto de backend utilizando IntelliJ IDEA y el servidor Tomcat.
  • Crear y configurar servlets y JSP para manejar solicitudes HTTP y generar contenido dinámico.

2. Implementación de Spring Core y Spring Boot:

  • Configurar un proyecto Spring Boot utilizando IntelliJ IDEA.
  • Crear y gestionar beans, y utilizar la inyección de dependencias para manejar las dependencias entre componentes.
  • Implementar módulos específicos de Spring, como Spring MVC para controladores y vistas, y Spring Data para acceso a datos.
  • Desplegar la aplicación utilizando Spring Boot para facilitar el proceso de configuración y despliegue.

3. Laboratorio 12:

  • Desarrollar un sitio web backend utilizando Java, Spring Core, y Spring Boot.
  • Configurar el proyecto y las dependencias necesarias en IntelliJ IDEA.
  • Implementar servlets, JSP, y componentes de Spring para manejar la lógica del servidor y las respuestas HTTP.
