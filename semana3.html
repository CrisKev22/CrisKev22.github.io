CSS Bootstrap y Tailwind

Definiciones


1. Librería CSS Bootstrap: Framework de diseño web que facilita la creación de sitios web responsivos y modernos mediante el uso de componentes y estilos predefinidos.

2. Layout y Components (Bootstrap): - Layout: Diseño y organización de los elementos en una página web utilizando la cuadrícula de Bootstrap y sus clases de diseño. - Components: Elementos reutilizables y estilizados, como botones, tarjetas, y formularios, que facilitan la creación rápida de interfaces de usuario.

3. Librería CSS Tailwind: Framework de utilidad-first para CSS que proporciona clases para diseñar rápidamente interfaces web sin necesidad de escribir CSS personalizado.

4. Tipografía (Tailwind): - Font: Tipos de letra y estilos tipográficos. - Line: Espaciado entre líneas de texto. - Text: Estilos de texto como color y alineación. - Background: Estilos de fondo, como colores y imágenes. - Borders: Estilos de bordes para elementos. - Effects: Efectos visuales como sombras y transformaciones. - Tables: Estilos para tablas. - Spacing: Espaciado en márgenes y relleno.

5. Layout (Tailwind): - Container: Contenedor centralizado para el contenido. - Columns: Diseño de columnas para organizar el contenido. - Break: Control de saltos y líneas. - Box: Diseño de cajas y contenedores. - Display: Control de la visibilidad y disposición de los elementos. - Position: Posicionamiento de los elementos en la página.

6. Flexbox (Tailwind): - Flex básico: Creación de contenedores flexibles. - Direction: Dirección de los elementos flexibles (filas o columnas). - Wrap: Ajuste de los elementos flexibles cuando no caben en una línea. - Grow: Expansión de los elementos flexibles. - Shrink: Reducción de los elementos flexibles.

7. Application UI (Tailwind): - Elements: Elementos de la interfaz de usuario como Avatars, Badges, Dropdowns, Button y Button Group. - Shells: Contenedores y estructuras de página. - Heading: Encabezados y títulos. - Data Display: Presentación de datos en la interfaz. - List: Listas y elementos de lista. - Form: Formularios y campos de entrada. - Feedback: Mensajes de retroalimentación para el usuario. - Navigation: Elementos de navegación como menús y barras. - Overlays: Elementos superpuestos como modales y popups. - Layout: Diseño general de la página. - Components: Componentes reutilizables y estilizados.

Procedimientos


1. Desarrollo de una página web Front-End con Bootstrap:

  • Crear una estructura básica de la página utilizando el sistema de cuadrícula de Bootstrap.
  • Incorporar componentes predefinidos de Bootstrap como botones, tarjetas y formularios.
  • Ajustar el diseño para que sea responsivo utilizando las clases de Bootstrap.

2. Desarrollo de una página web Front-End con Tailwind:

  • Diseñar la página utilizando las utilidades de Tailwind para la tipografía, espaciado y fondos.
  • Organizar el layout utilizando las clases de contenedores, columnas y posicionamiento de Tailwind.
  • Aplicar estilos de Flexbox para gestionar la distribución de los elementos en la página.

3. Práctica Calificada 01:

  • Desarrollar un sitio web completo utilizando un framework (Bootstrap o Tailwind), implementando los conceptos aprendidos en el curso.
  • Aplicar las técnicas de diseño y componentes aprendidos para crear una interfaz de usuario funcional y estilizada.
