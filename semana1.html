Fundamentos de las Tecnologías Web

Definiciones


1. Fundamentos de las tecnologías web: Conjunto de conocimientos básicos necesarios para entender el desarrollo y funcionamiento de aplicaciones web, incluyendo protocolos, lenguajes de programación y estructuras de red.

2. Tecnologías web básicas: Incluyen HTML para la estructura de las páginas, CSS para el estilo y diseño, y JavaScript para la interactividad.

3. DNS (Domain Name System): Sistema que traduce los nombres de dominio legibles por humanos (como www.ejemplo.com) en direcciones IP numéricas que las computadoras utilizan para identificarse entre sí en la red.

4. Protocolo TCP/IP: Conjunto de reglas que permiten la transmisión de datos en internet, asegurando que los datos lleguen correctamente a su destino.

5. Protocolo HTTP (HyperText Transfer Protocol): Protocolo de comunicación utilizado para la transferencia de datos en la web.

6. Roles en el desarrollo de aplicaciones para Internet:

  • Desarrollador frontend: Encargado de la parte visual e interactiva de una aplicación web.
  • Desarrollador backend: Se ocupa de la lógica del servidor, bases de datos y la integración del frontend.
  • Desarrollador fullstack: Combina las habilidades del desarrollador frontend y backend, pudiendo trabajar en todas las áreas del desarrollo web.

7. Visual Studio Code (VSC): Editor de código fuente desarrollado por Microsoft, utilizado para desarrollar aplicaciones web, entre otras cosas. Incluye características como la depuración, control de versiones y soporte para extensiones.

Procedimientos


1. Instalación y configuración de Visual Studio Code:

  • Descargar Visual Studio Code desde la página oficial.
  • Instalar el software y configurarlo según las preferencias personales.
  • Explorar los elementos de la interfaz de usuario, como la barra lateral, el explorador de archivos y la terminal integrada.

2. Uso de extensiones en Visual Studio Code:

  • Instalar extensiones útiles para el desarrollo web, como Prettier para el formateo de código y Live Server para ver los cambios en tiempo real.

3. Práctica de atajos de teclado:

  • Familiarizarse con los atajos de teclado más comunes en VSC para mejorar la eficiencia, como Ctrl+P para abrir rápidamente archivos y Ctrl+Shift+P para acceder a los comandos.
