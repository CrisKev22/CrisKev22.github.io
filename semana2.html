Fundamentos de las Tecnologías Web - Páginas con HTML y CSS

Definiciones


1. Lenguaje Marcado de Hipertexto (HTML): Lenguaje de marcado utilizado para crear y estructurar el contenido de las páginas web. Define elementos como encabezados, párrafos, enlaces y otros componentes estructurales.

2. Lenguaje Marcado Extensible (XML): Lenguaje de marcado utilizado para almacenar y transportar datos. Es extensible y permite definir etiquetas personalizadas.

3. Estándar Open Web Platform: Conjunto de tecnologías abiertas que permiten el desarrollo de aplicaciones web, incluyendo HTML, CSS, y JavaScript.

4. Estructura HTML y XML: HTML y XML utilizan una estructura jerárquica de etiquetas para organizar el contenido y los datos. HTML es utilizado para la presentación, mientras que XML se usa para el intercambio de datos.

6. Árbol DOM (Document Object Model): Representación en memoria de la estructura de un documento HTML o XML, donde cada nodo representa un elemento del documento.

7. Renderizado de Árbol: Proceso mediante el cual el navegador convierte el árbol DOM en una representación visual de la página web.

8. Ciclo de Vida de una Página Web: Fases por las que pasa una página web desde su carga inicial hasta su cierre, incluyendo la carga de recursos y la interacción con el usuario.

9. Optimización en motores de búsqueda (SEO): Técnicas y prácticas utilizadas para mejorar la visibilidad de una página web en los resultados de búsqueda de los motores de búsqueda.

10. Compatibilidad entre navegadores: Garantizar que una página web funcione correctamente en diferentes navegadores web, ajustando el diseño y el comportamiento según sea necesario.

11. Hojas de Estilos (CSS): Lenguaje de hojas de estilo utilizado para definir la presentación visual de una página web, incluyendo el diseño, colores y fuentes.

12. Estándar CSS3: Versión más reciente del lenguaje CSS, que incluye nuevas características como flexbox, grid layout, y mejoras en la sintaxis.

Procedimientos


1. Construcción de una página web con HTML y XML:

  • Crear una estructura básica de HTML utilizando etiquetas principales.
  • Utilizar XML para almacenar y transportar datos según sea necesario.
  • Verificar la correcta visualización y funcionalidad en diferentes navegadores.

2. Diseño de la página web con CSS:

  • Aplicar estilos utilizando CSS3, incluyendo la definición de tamaños, colores y fuentes.
  • Utilizar flexbox y grid layout para crear diseños responsivos y fluidos.
  • Configurar el diseño de la página para adaptarse a diferentes tamaños de pantalla y dispositivos.

3. Optimización para SEO y compatibilidad entre navegadores:

  • Implementar prácticas de SEO para mejorar la visibilidad de la página web.
  • Probar la compatibilidad de la página en diferentes navegadores y realizar ajustes según sea necesario.
