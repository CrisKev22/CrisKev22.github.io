Desarrollar una página web Front - End con Javascript

Definiciones


1. Motores JavaScript: Entornos de ejecución para el código JavaScript en el navegador o en el servidor, como V8 en Chrome y Node.js.

2. Variables y operadores: - Variables: Espacios de almacenamiento para datos, definidos con `var`, `let`, o `const`. - Operadores: Símbolos que realizan operaciones sobre variables y valores, como `+`, `-`, `*`, `/`, `==`, `===`.

3. Tipos de datos: - Números: Valores numéricos que pueden ser enteros o de punto flotante. - Strings: Cadenas de texto, definidos entre comillas simples o dobles. - Objetos: Colecciones de propiedades y métodos, definidos con llaves `{}`. - Booleanos: Valores `true` o `false`.

4. Estructuras de control y bucles: - Estructuras de control: `if`, `else`, `switch` para decidir qué bloques de código ejecutar. - Bucles: `for`, `while`, `do-while` para repetir bloques de código.

5. Funciones: - Sintaxis: `function nombre(params) { /* código */ }`. - Invocación: Llamar a una función con `nombre()`. - Funciones anónimas: Funciones sin nombre, a menudo usadas como argumentos. - Función objeto: Funciones que se comportan como objetos. - Funciones flecha: Sintaxis concisa para funciones: `(params) => { /* código */ }`. - Funciones auto invocadas: Funciones que se ejecutan inmediatamente después de su definición: `(function() { /* código */ })();`. - Funciones closures: Funciones que recuerdan el contexto en el que fueron creadas.

6. Clases Number, String, Math: - Number: Objeto para trabajar con números. - String: Objeto para manipular cadenas de texto. - Math: Objeto con métodos y propiedades matemáticas, como `Math.random()` y `Math.PI`.

7. Arrays unidimensionales y multidimensionales: - Arrays unidimensionales: Listas de elementos en una sola dimensión. - Arrays multidimensionales: Arrays dentro de arrays, creando matrices.

8. Objetos Map y Set: - Map: Colección de pares clave-valor donde las claves pueden ser de cualquier tipo. - Set: Colección de valores únicos, sin duplicados.

9. Manejo del DOM con JavaScript: - Métodos y propiedades del DOM: Manipulación de elementos HTML y atributos. - Elementos y estilos con JavaScript: Cambio de contenido y estilo de elementos HTML. - Animaciones y gráficos con Canvas: Creación de gráficos y animaciones usando el elemento ``.

Procedimientos


1. Desarrollo de una página web Front-End con JavaScript:

  • Crear variables y operadores para manejar datos y realizar operaciones básicas.
  • Implementar estructuras de control y bucles para gestionar la lógica de la página.
  • Definir y utilizar funciones para modularizar el código, incluyendo funciones anónimas, flecha y closures.
  • Manipular el DOM para cambiar el contenido y estilo de los elementos HTML.

2. Uso de clases y objetos en JavaScript:

  • Utilizar las clases Number, String y Math para manejar datos numéricos y cadenas de texto.
  • Trabajar con arrays unidimensionales y multidimensionales para almacenar y organizar datos.
  • Aplicar Map y Set para gestionar colecciones de datos con claves y valores únicos.

3. Manejo de Canvas y animaciones:

  • Crear gráficos y animaciones utilizando el elemento y su API.
  • Desarrollar una animación interactiva como parte del Laboratorio 04.
