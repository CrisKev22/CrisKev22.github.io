Trabajos Realizados

Semana 1

Fundamentos de las tecnologías web 8 al 12 de abril

En la primera semana conocimos cuales son las tecnologías web, tambien vimos conceptos de desarrollador backend y frontend. Hicimos la instalación de Visual Studio Code, uso de extensiones y atajos de teclado con Emmet.


Ver página

Semana 3

CSS Bootstrap y Tailwind 22 al 26 de abril

En la semana 3 se desarrollo la creación de paginas Web con el uso de librerias, en este caso se usaron 2 librerias: Librería de CSS Bootstrap y Librería CSS Tailwind. El uso de las librerias optimiza el tiempo para la creación de las páginas.


Ver página

Semana 5

Aplicación FrontEnd con Framework JS 6 al 10 de mayo

En esta semana se hizo la estructura y operación de Framework, realizamos la gestión de dependencias y el desarrollo basado en componentes - JSX / PROPS / CHILDREN. La utilización de estilos en Framework JS, pasar datos con props y children.


Ver página

Semana 2

Página con HTML y XML 15 al 19 de abril

En la segunda semana conocimos las etiquetas principales de HTML5, el ciclo de vida de una página web, optimización en motores de busqueda SEO y la compatibilidad con los navegadores. Se diseño pagina web con Hojas de Estilo.



Ver página

Semana 4

Página Web con Javascript 29 de abril al 3 de mayo

En esta semana conocimos más acerca de Javascript, asi como sus variables y operadores, estructura de controles y bucles. Se hizo el DOM con Javascript y Canvas, se hizo desarrollo de una animación con Javascript.


Ver página

Semana 6

Desarrollo de Aplicaciones con Eventos, Condicionales 13 al 17 de mayo

En esta semana se reviso el uso de Eventos, Renderizado condicional, Renderizado Iterativo, Formularios. El consumo de API's. El evento y renderizado condicional e iterativo con datos de una API.


Ver página

Semana 7

Uso de Hooks 20 al 24 de mayo

En la semana 7 vimos el uso de Hooks y cuales son funcionalidades, estudiamos y practicamos con los Hooks mas conocidos: UseState / UseEffect / UseContext / UseRef / UseReducer / UseCallback / useMemo Hooks Personalizados


Ver página

Semana 10

Next JS - CSR / SSR 10 al 14 de junio

En la semana 10 estudiamos el tema de Next JS, realizamos la creación de un proyecto, tambien el routing en Next, los diferentes estilos que maneja, el uso de formularios y el uso de Typescript. Realizamos ejercicios practicos para ello.


Ver página

Semana 12

Desarrollo Backend con Spring Core y Spring Boot 24 al 28 de junio

En la semana 12 desarrollamos el Backend, la arquitectura de aplicaciones web, servidores web, funcionamiento del server side, lenguaje y framework backend. Java y Jakarta. IntelliJ Idea, Tom Cat.



Ver página

Semana 9

Next JS - CSR / SSR 3 al 7 de junio

En la semana 9 revisamos lo que es Next JS - CSR / SSR, revisamos la creación del proyecto, tambien el routing en Next, el ingeniero nos enseño los estilos, nos enseño la creación de formularios, tambien nos enseño el lenguaje Typescript.



Ver página

Semana 11

Server Side Props - Next Auth 17 al 21 de junio

En la semana 11 revisamos todo lo que es el Server SideProps, el ingeniero nos enseño la instalación, configuración, obtener información de la sesión, proveedor, credenciales, callbacks, verificar credenciales. Creación de usuario basado en OAuth, login personalizado, Middlewares.


Ver página

Semana 13

Desarrollar API Rest con Spring 1 al 5 de julio

En la semana 13 aprendimos a desarrollar un API Rest con Spring, el conocimiento de protocolo HTTP, MVC, nos enseño acerca de la Arquitectura API Rest, conceptos basicos, funcionamiento e importancia y rol que tiene en la programación hoy en dia.
