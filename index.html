Trabajos Realizados

Semana 1

Fundamentos de las tecnologías web 8 al 12 de abril

En la primera semana conocimos cuales son las tecnologías web, tambien vimos conceptos de desarrollador backend y frontend. Hicimos la instalación de Visual Studio Code, uso de extensiones y atajos de teclado con Emmet.

Semana 3

CSS Bootstrap y Tailwind 22 al 26 de abril

En la semana 3 se desarrollo la creación de paginas Web con el uso de librerias, en este caso se usaron 2 librerias: Librería de CSS Bootstrap y Librería CSS Tailwind. El uso de las librerias optimiza el tiempo para la creación de las páginas.

Semana 5

Aplicación FrontEnd con Framework JS 6 al 10 de mayo

En esta semana se hizo la estructura y operación de Framework, realizamos la gestión de dependencias y el desarrollo basado en componentes - JSX / PROPS / CHILDREN. La utilización de estilos en Framework JS, pasar datos con props y children.

Semana 2

Página con HTML y XML 15 al 19 de abril

En la segunda semana conocimos las etiquetas principales de HTML5, el ciclo de vida de una página web, optimización en motores de busqueda SEO y la compatibilidad con los navegadores. Se diseño pagina web con Hojas de Estilo.

Semana 4

Página Web con Javascript 29 de abril al 3 de mayo

En esta semana conocimos más acerca de Javascript, asi como sus variables y operadores, estructura de controles y bucles. Se hizo el DOM con Javascript y Canvas, se hizo desarrollo de una animación con Javascript.

Semana 6

Desarrollo de Aplicaciones con Eventos, Condicionales 13 al 17 de mayo

En esta semana se reviso el uso de Eventos, Renderizado condicional, Renderizado Iterativo, Formularios. El consumo de API's. El evento y renderizado condicional e iterativo con datos de una API.
